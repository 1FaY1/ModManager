# 🛠 Mod Manager Pro v1.7
Легкая, быстрая и прозрачная утилита для управления модами Minecraft через **Modrinth API**.
---
## ⚙ Основные возможности
* **Мгновенный поиск:** Находите любые моды, доступные на платформе Modrinth.
* **Проверка обновлений:** Сканирование папки `mods` и определение устаревших файлов с помощью хеширования SHA1.
* **Обновление в один клик:** Скачивание последних версий модов прямо в папку с игрой.
* **Портативность:** Один исполняемый файл (.exe), не требующий установки.
* **Многопоточность:** Высокая скорость работы за счет параллельных запросов к API.
---
## 🛡 Безопасность и прозрачность
Проект имеет **открытый исходный код**:
* **Официальный API:** Приложение общается только с серверами Modrinth.
* **Чистота:** Без рекламы, слежки и лишних процессов.
---
## 🚀 Как пользоваться
1. **Скачайте** актуальный файл из раздела [Releases](https://github.com/1FaY1/ModManager/releases/latest).
2. **Запустите** программу.
3. **Нажмите** кнопку "📂 Выбрать сборку".
4. **Перенесите или укажите** путь к вашей папке `mods`.
5. **Нажмите** "🔄 Проверить обновления" или **"⬇️ Обновить всё"**, чтобы синхронизировать сборку.
---
## 📄 Лицензия и условия
Этот проект создан для образовательных и личных целей.
* **Ограничено:** Копирование исходного кода в коммерческих целях или перепродажа модифицированных версий без разрешения автора.
